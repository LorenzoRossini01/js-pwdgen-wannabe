# js-pwdgen-wannabe

## Traccia

```
Ciao ragazzi,
esercizio di oggi: (insicurissimo) password generator
nome repo: js-pwdgen-wannabe

Chiedi all'utente il suo nome,
poi chiedi il suo cognome,
poi chiedi il suo colore preferito
Infine scrivi sulla pagina nomecognomecolorepreferito23
```

## Svolgimento

```
- Step 1 - Chiedo all'utente il suo Nome
    - trasformo il dato in input in una variabile per poterla utlizzare successivamente
```

```
- Step 2 - Chiedo all'utente il suo Cognome
    - trasformo il dato in input in una variabile per poterla utlizzare successivamente
```

```
- Step 3 - Chiedo all'utente il suo colore preferito
    - trasformo il dato in input in una variabile per poterla utlizzare successivamente
```

```
- Step 4 - Stampo la password ottenuta dalla concatenazione delle informazioni date dall'utente
```
